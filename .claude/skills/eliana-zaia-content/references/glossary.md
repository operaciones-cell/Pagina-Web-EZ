# Glossary — diccionario de marca

Términos técnicos, ingredientes destacados, palabras de oficio. Usar cuando aporten precisión; no para decorar.

---

## Términos de oficio

| Término | Significado | Cuándo usarlo |
|---|---|---|
| **Oficio** | Conjunto de conocimientos prácticos transmitidos | Eje central de la voz — usar libremente |
| **Pieza** | Cada postre individual | Reemplazo editorial de "producto" |
| **Carta** | Conjunto de piezas disponibles | Reemplazo editorial de "catálogo" o "menú" |
| **Estación** | Periodo del año (no "temporada") | Más sutil, evoca lo cíclico |
| **Lote** | Producción acotada de una pieza | Para piezas limitadas |
| **Receta** | Fórmula con autoría | Implica decisiones, no solo ingredientes |
| **Maestría** | Dominio de una técnica | Solo cuando hay sustento, no como adjetivo |
| **Cocina** | El espacio físico donde se trabaja | Más cálido que "taller" |
| **Mesa** | El destino del postre (la mesa del cliente) | "Para tu mesa" — mejor que "para ti" |
| **Casa** | El negocio del aliado (hotel, restaurante) | "Trabajamos con casas que…" |

## Tradición latina (ingredientes y técnicas colombianas)

| Término | Notas |
|---|---|
| **Café de altura** | Café cultivado en zonas altas — diferenciador colombiano |
| **Panela** | Azúcar sin refinar — tradición latinoamericana, valorar por su carácter |
| **Cacao fino de aroma** | Cacao colombiano premium — usar el término completo |
| **Frutas de altura** | Frutas de la sabana cundiboyacense (uchuva, curuba, lulo, gulupa) |
| **Maracuyá** | No "fruta de la pasión" en copy de marca |
| **Sabana de Bogotá** | Región geográfica relevante para el origen |
| **Zipaquirá** | Siempre con tilde, siempre el nombre completo |

## Técnica italiana

| Término | Notas |
|---|---|
| **Mascarpone** | Queso italiano fresco — base del tiramisú |
| **Pasticceria** | Pastelería italiana — usar solo si aporta contexto preciso, no decorativo |
| **Veneto** | Región del tiramisú original |
| **Sicilia** | Origen del cannoli, granita |
| **Génesis (génoise)** | Bizcocho clásico — preferir "génoise" si el contexto lo permite |
| **Pâte sablée** | Masa quebrada con manteca — usar el término técnico cuando habla a B2B |
| **Frangipane** | Crema de almendras |
| **Crémeux** | Crema sedosa, estable — término técnico |
| **Glaçage** | Glaseado fino brillante |
| **Témperaje** | Cristalización del chocolate — proceso, no decoración |
| **Confitado** | Cocción lenta en azúcar — se puede usar de fruta |

**Regla**: términos técnicos italianos/franceses solo cuando aportan precisión real al lector. Si "bizcocho" funciona, no escribir "génoise". El registro técnico es para descripciones de producto profundas o copy B2B, no para hero del home.

## Términos a evitar (anti-glosario)

| No usar | Por qué | Reemplazo |
|---|---|---|
| Delicia | Adjetivo vacío y gastado | Describir qué la hace particular |
| Exquisito | Vacío | Mostrar la cualidad concreta |
| Único | Hueco si no se sustenta | "El único que…" + razón |
| Premium (como adjetivo de copy) | Es categoría de mercado, no descripción | "Artesanal", "del oficio" |
| Gourmet | Connotación gastada | "De cocina" o no describir |
| Dulce (como sinónimo de postre) | Genérico, infantil | "Postre" o "pieza" |
| Sabor único | Frase hecha | Describir el sabor real |
| Made with love | Cliché global | "Hecho con tiempo" o no decir |
| Auténtico | Sospechoso por su uso | "Tradicional" si aplica |
| Casero | Connota informal — Eliana es boutique | "Artesanal" |
| Artesanal de verdad | Defensivo | Solo "artesanal" |
| Selección | Ambiguo de e-commerce | "Carta" o "lo que ofrecemos" |
| Catálogo | Frío | "Carta" |
| Producto / productos | Plano | "Pieza" / "carta" / nombrar el postre |
| Cliente (en copy público) | Transaccional | "Quienes nos buscan", "tu mesa" |
| Tienda | Connota retail | "Carta", "puntos de venta" |
| Sucursal | Corporativo | "Punto de venta" o "casa aliada" |

## Palabras ancla recomendadas (para titulares e italics)

Estas funcionan especialmente bien como palabras de énfasis (italic gold) en titulares serif:

- **conversación**, **paciencia**, **tradición**, **contemporáneo**
- **oficio**, **maestría**, **autoría**
- **memoria**, **estación**, **raíz**
- **detalle**, **autoría**, **encuentro**
- **respiro**, **pausa**, **tiempo**

Evitar como ancla: palabras genéricas (calidad, sabor, amor, único, especial).

## Variaciones de marca

Cómo escribir el nombre según contexto:

| Contexto | Forma |
|---|---|
| Logo / encabezado | "Eliana *Zaia*" (Zaia en italic gold) |
| Cuerpo de texto | "Eliana Zaia" (sin formato especial) |
| Después de la primera mención en un mismo párrafo | Evitar repetir; usar "la casa", "el oficio" o sujeto tácito |
| Footer / legal | "Eliana Zaia" |
| Email / firma | "Eliana Zaia · Postres Artesanales" |
| URL / dominio | elianazaia.com |
| Handle redes | @elianazaia |

---

## Tratamiento

- **"Tú" y "vos"**: el sitio actual usa "tú" para audiencia colombiana. Mantener consistencia. Nunca mezclar en una misma página.
- **"Nosotros"**: la marca habla en primera persona plural. "Cultivamos", "trabajamos", "respetamos". Nunca tercera persona ("Eliana Zaia es una empresa que…").
- **"Eliana" como persona**: solo en sección de oficio o about cuando hablar de ella aporta autoridad. No personalizar todo el copy con su nombre.
